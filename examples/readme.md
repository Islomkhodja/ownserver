# ownserver example

Bu example'da knex, pg, jayson (json rpc) ishlatilgan..

## install 
`npm install`

### main page
```
GET /
```
### get all persons
`GET /person`

### add person/s
`POST /person`

### get person by id
`GET /person/{id}`

### update person by id
`PUT /person/{id}`

### delete person by id
`DELETE /person/{id}`

### JSON RPC
`POST /api`

### json rpc method
`getList`
